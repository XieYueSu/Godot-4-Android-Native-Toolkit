# 🚀 Godot 4 Android Native Toolkit
### Camera • QR Scanner • Ads • Media • File Picker • Share • Notifications • Video Player

A complete **Android native plugin toolkit for Godot 4**, designed to let you access powerful Android system features directly from GDScript—**no Java/Kotlin required**.

---

## 📦 What is this?

This is a curated collection of lightweight Android native plugins for Godot 4.

Instead of writing native Android code, you can directly call simple GDScript APIs to use system features like:

- Camera access & photo capture
- QR code scanning
- AdMob ads integration
- File & image picking
- Native sharing system
- Local notifications
- Video playback (online + local)

---

## ✨ Key Features

- ✅ **Zero Native Coding** – No Java/Kotlin required
- ⚡ **Lightweight** – Uses Android built-in APIs only
- 🎮 **Godot 4 Ready** – Fully compatible with stable Godot 4.x
- 🔌 **Drop-in Integration** – Just enable plugin and start using
- 📡 **Consistent API Design** – Same usage pattern across all modules
- 📱 **Android Native Performance** – No heavy external SDK dependencies

## 📦 Included Plugins

---

### 📷 1. Camera Plugin
Capture photos directly from Android camera inside Godot.

**Features:**
- Native camera preview in Godot viewport
- One-click photo capture
- Save images to local storage
- Real-time rendering with low overhead

---

### 📷 2. QR Scanner Plugin
Fast native QR / barcode scanning.

**Features:**
- Camera-based scanning
- Instant result callback to GDScript
- Lightweight native implementation

---

### 📢 3. AdMob Ads Plugin
Integrate Google AdMob easily.

**Features:**
- Banner / Interstitial / Rewarded ads
- Simple GDScript API
- Full ad lifecycle callbacks

---

### 📁 4. File Picker Plugin
Native Android file selection dialog.

**Features:**
- Pick images, videos, audio, documents
- Folder support
- Returns file path directly to Godot

---

### 🖼️ 5. Image Picker & Editor Plugin
Pick and edit images from gallery.

**Features:**
- Gallery image selection
- Image compression
- Crop & circular avatar cropping
- Zoom preview support

---

### 🔔 6. Notification Plugin
Local Android notifications system.

**Features:**
- Schedule / cancel notifications
- Custom icons
- Click callback support
- Immediate or delayed alerts

---

### 🔗 7. Share Plugin
Native Android sharing system.

**Features:**
- Share images, videos, files, URLs
- Supports WhatsApp, Telegram, Email, etc.
- Native Android share sheet

---

### 🎬 8. Video Player Plugin
Native Android video playback using hardware acceleration.

**Features:**
- Play MP4 / HLS (m3u8)
- Online streaming support
- Full controls (play, pause, seek, speed, loop)
- Fullscreen support
